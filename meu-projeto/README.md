## AULA 1
### Criação da aplicação\

abrir o prompt de comando\
executar o comando dentro de uma pasta sua:\
npm create vite@latest meu-projeto -- --template react \
  
para executar tem que entrar na pasta: \
cd meu-projeto  \
\
depois executar o comando:\
npm install\
code . \
npm run dev \
